||| Starting the System

dspm-archive venue launch · top to bottom · don't skip a box.

1

Power up

norns + the laptop on the same WiFi.

2

Pick the script on norns

SELECT → dspm_archive.

3

Get norns' IP address

On norns: SYSTEM → WIFI (e.g. 10.0.0.35).

4

Open PowerShell in the repo

The dspm-archive folder — the root, not bridge\.

5

Run the launch command

./start-venue.ps1 -NornsHost 10.0.0.35

Swap in your real IP. Add -Tunnel to put it online with an audience QR.

6

Read the printed URLs

It lists the audience link and the performer link.

7

Hand out the audience URL

Keep the performer.html?token=… page to yourself.

8

When done: press Ctrl-C

Stops the bridge (and tunnel) and saves the session log. Don't just close the window.

!

First time only — once per laptop

Run PowerShell as Administrator and allow port 8081 so phones can reach you (New-NetFirewallRule … -LocalPort 8081). Skip if already done.
